Modulo Profesor

Guiones de prueba para El modulo profesor de la aplicación

Prueba N°1

Área de la prueba: Windows

Objetivo de la prueba: Verificar que la aplicacion registre un profesor de forma correcta

Requerimientos: Haber iniciado sesion como administrador o secretaria

    Acciones

    Descripción:

  • En el menu seleccione la opcion registrar profesor
  • Debe mostrarse el formulario de registro de profesor
  • Ingrese los datos solicitados de forma valida
  • Se debe presionar el boton registrar para enviar los datos

    Resultado esparado:

  • Se debe mostrar el mensaje de profesor registrado con exito

Prueba N°2

Área de la prueba: Windows

Objetivo de la prueba: Verificar que la aplicacion modifique un profesor de forma correcta

Requerimientos: Haber iniciado sesion como administrador o secretaria y que exista el registro del profesor

    Acciones

    Descripción:

  • En el menu seleccione la opcion actualizar profesor
  • Debe mostrarse el formulario para actualizar profesor
  • Ingresar la identificacion del profesor en el campo de busqueda
  • Se debe presionar el boton buscar para buscar la informacion del profesor relacionada a esa identificación
  • Se deben llenar los campos con la informacion encontrada
  • Debe modificar el campo o campos que se desean cambiar
  • Se debe presionar el boton actualizar para guardar la informacion

    Resultado esparado:

  • Se debe mostrar el mensaje de informacion modificada con exito
